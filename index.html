    Online C Code Editor


    Links
    Application Link: http://13.229.230.199/
    GitHub Link: https://github.com/relientm96/online_code_editor

    About

    Made a Full Stack online C Code integrated development environment web application where users can log on to run C programs. Backend server created using C++ and POCO C++ Libraries and deployed onto an EC2 Windows server instance on the AWS cloud. Simple code samples are also provided to users.

    How it works

    When a user creates a project from the login screen, a .c file is created on the server's file system. Once the user is happy to compile their program, they would click 'compile' which sends the user's code on the editor as a text payload to the server. The text is copied into the .c file and the GCC compiler compiles the file. The server then responds by sending either the error output if there is an error in compilation or the successful compiled file output to the browser. Once the user closes the site, an automatic "DELETE " request is made to the server to delete the user's .c and .exe files to clear space.

    Features

    • C++ Backend with GCC 6.3.0 C compiler
    • FrontEnd user code text editor interface
    • Sample simple C programs provided
    • Registration feature for users

    Utilized

    • POCO C++ Libraries
    • CodeMirror as code editor
    • HTML, CSS, Javascript and Bootstrap for front end web design
    • RESTAPI calls to backend and handling of JSON payload.

    Seven Segment on Web

    Links
    GitHub Link: https://github.com/relientm96/SevenSegment

    About

    An arduino based project that allows for Seven Segment Digit control over a Web interface.
    The ESP8266 is used as a Web Server that sends out a html webpage for user interaction.
    The Arduino UNO is used to collect received serial data (transmitted through UART) from the ESP8266 and displays the number onto the double digit seven segment display.

    Available Modes

    • Potentiometer Mode: Control number to display by reading analog pin connected to Potentiometer.
    • Auto Counter: Arduino automatically increments from 0-99 and displays it.
    • Web Server Control: Utilizes the input number on Web Interface which it sent to the ESP8266 followed by the UNO for display.

    Digital Signal Oscilloscope

    Video demonstration of the Web Interface

    Links
    GitHub Link: https://github.com/relientm96/DSO2019

    About

    This project was made to gain exposure to electronic system design guided by the ELEN90053 Electronic System Design subject at the University of Melbourne. Personally, I added an external IOT Circuit made with an ESP8266 to enable web communication. Additionally, I created a front end web interface to view the amplitude and Fast Fourier transform plot of the signal.

    Features

    • Printed Circuit Board designed in Altium Designer.
    • Atmega16 microcontroller programmed in C.
    • ESP8266 WiFi chip with OLED Screen Display.
    • Web Interface displaying waveform and discrete fourier transform plots, developed using Express.js, HighCharts and Materialize.
    • Created a 3D printed encolusre using SolidWorks.

    How it works

    Input signal is first pre-processed during the input stage of the device. Signal goes through impedance scaling, filtering and level shifting before being fed into the Analog to Digital converter stage. The signal is then sampled as an 8-bit value at sampling rates 40 MHz, 20 MHz or 10 Mhz chosen by the user. 8-bit sampled values are then stored in 2 8-bit buffers which is then later transfered to the micro-controller and sent to the web browser and computer GUI. The CPLD acts as a decision making device that routes signals based on logic gates at the chosen high sampling rates.

    Arduino Audio Processing Shield

    Picture of designed audio shield

    Links
    Schematics Link: https://easyeda.com/matthewyfy/arduino-audio-processor-shield
    GitHub Link: https://github.com/relientm96/unoAudioBoard

    About

    Inspired by my Signal Processing class, I created a real time digital audio processing shield that can be mounted onto an Arduino UNO board. The board takes in audio and adds in digital audio effects in real time. Input audio is first pre-processed at the input stage before being fed into the Arduino's Analog to Digital converter. The ADC represents the signal as an 8-bit digital value and is then given the chosen effect. After processing, the Arduino outputs the processed audio as an 8-bit value into a digital to analog converter to produce the sound with desired effect.

    Programmed Effects

    • Flanging
    • Phasing
    • Chorusing
    • Distortion
    • Tremolo
    • Bit Crusher

    Features

    • An op-amp based pre-amp circuit.
    • OLED Display, showing the current effect applied to the input audio.
    • Output audio played using an 8-bit R2R resistor ladder DAC.
    • Can be connected to an electric guitar for digital audio effects.

    Arduino Waveform Signal Generator

    Video Demonstration of Arduino Nano Prototype with speakers and oscilloscope (WARNING Audio may be loud!)

    Links
    GitHub Link: https://github.com/relientm96/digitalSigGen

    About

    I created this project to understand and learn more about how digital audio synthesis. This digital waveform generator is capable of producing sine, square, sawtooth and triangle waveforms from an Atmega328p micro-processor. I made two versions, with one being an arduino nano prototype and the other an Arduino Uno shield. Both consists of an R2R resisto ladder Digital to Analog converter and buffed with an op-amp voltage follower. It can also play back PCM encoded audio.

    Features

    • Able to generate Sine, Triangle, Square, Sawtooth waveforms.
    • Capable of playing PCM encoded audio.
    • 8-bit R2R Digital to Analog converter using resistors and an op-amp voltage follower.
    • Two versions, one being the Arduino UNO shield and the other an Arduino Nano prototype.

    Utilized

    • Digital Signal Processing in C/C++.
    • Arduino IDE environment.
    • Pulse Code Modulation encoding.
