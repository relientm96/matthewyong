Cloud and Cluster Computing Project



Worked in a group of five to create a cloud based web application that performs twitter harvesting and large data analysis on tweets from throughout Australia. The data was harvested using Python, stored on a CouchDB cluster and hosted on the NeCTAR Research Cloud. I worked on the automatic deployment of the system to any remote host using Ansible and bash scripts.
(View project repository here )
Key Features
• Three node CouchDB Cluster.
• Flask Backend and Apache2 for deployment.
• Ansible and Bash Script for Automation Deployment.
• Python based tweet streamer using Tweepy library.
• Deployed onto three virtual machines (Each on Ubuntu 18.04) on the NecTAR Research Cloud.
