A class project involving the design and development of an electronic embedded system. Altium was used to design the Printed Circuit Board and AVR Studio was used to program the ATMEGA-16 micro-controller in C.

Games:
• Flappy Bird (simplified)
• Etch A Sketch
• Pong (with two level AIs)
• Monty Hall

Features:
• Screen brightness control using PWM.
• Four movement buttons and two action buttons.
• Battery level detector.
• Serial Peripheral Interface communication between micro-controller and externals.

Cloud and Cluster Computing Project



Worked in a group of five to create a cloud based web application that performs twitter harvesting and large data analysis on tweets from throughout Australia. The data was harvested using Python, stored on a CouchDB cluster and was hosted on the NeCTAR Research Cloud. I worked on the automatic deployment of the system to any remote host using Ansible and bash scripts.

Tools:
• Docker
• CouchDB (for MapReducing data)
• Ansible
• Python (Twitter Harvesting)
• Flask
• Apache2
